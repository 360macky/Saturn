## 🚀 Deployment
💻 Once you have cloned or directly downloaded this repository: Open a terminal, go to the project 
folder and run:

📦 Install the dependencies

```bash
npm install
```
🚀 Start the deployment website
```bash
npm start
```
🎉 **Great! Start using or developing in Saturn project.**
